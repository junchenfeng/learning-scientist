# 两样本结对均值检验

## 原理
一个较理想的随机实验对于每个学生有两个观测。这种数据结构被称为结对。

与此相比，如果随机分配是在两个班级，那么这两个班级均值的比较不是结对数据。

结对数据的一个很大好处是可以把两样本均值检验变成单样本检验

H0: mu_1 = mu_2 -> H0: mu_1 - mu_2 = 0

这样我们可以用之前的单样本t test来完成。
